Verrous PostgreSQL : PostgreSQL utilise un système de verrous multi-niveaux. Les verrous de table sont acquis automatiquement selon la commande SQL. La matrice de compatibilité détermine si deux verrous peuvent coexister. Quand un verrou est bloqué, un wait-for graph se forme. Si un cycle est détecté, PostgreSQL annule une transaction pour résoudre le deadlock.
